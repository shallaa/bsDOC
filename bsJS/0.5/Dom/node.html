는 #test의 자식이 됨 bs.dom("#test").$(">","
"); //

"); ``` #### # 자손 요소 컨트롤 ``` bs.Dom(xxx).S( '>인덱스>인덱스>인덱스>...' ); ``` * ">인덱스>인덱스>인덱스" : 특정 자손으로 손쉽게 드릴다운할 수 있음. * 자손요소 자체를 얻거나 키를 얻는 기능만 지원 ```javascript //2번자식의 3번자식(인덱스는 0부터) var element = bs.Dom("#test").S(">2>3"); //2번자식의 3번자식의 html var data = bs.Dom("#test").S(">2>3", "html"); ```
